Detailed Description

Header file defining C aggregator functions for SEPA.

Author
Francesco Antoniazzi franc.nosp@m.esco.nosp@m..anto.nosp@m.niaz.nosp@m.zi@un.nosp@m.ibo..nosp@m.it
Date
12 Apr 2017

This file contains the functions useful to build-up a SEPA aggregator. We provide function for querying, subscribing and unsubscribing and updates. Requires libcurl, libwebsockets, jsmn. You should not include it in your code only if you are writing a plain consumer or aggregator. In this case, it is strongly suggested to import the related header file.
