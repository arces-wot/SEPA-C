Detailed Description

struct secure_client (aka sClient) is the data structure we use to store sepa registration items for a client. You shouldn't modify its content, though it is possible that you need to read it. In particular, you might need the JSON Web Token which is stored as a string within the field JWT.

See also
https://tools.ietf.org/html/rfc7519
