Detailed Description

Header file defining C producer functions for SEPA.

Author

Date

This file contains the functions useful to build-up a SEPA producer. We provide function for updates. Requires libcurl. You should not include it in your code only if you are writing a plain consumer or aggregator. In this case, it is strongly suggested to import the related header file.
